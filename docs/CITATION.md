# Citation

```verbatim
To cite package ‘altdoc’ in publications use:

  Bacher E (2023). _altdoc: Package Documentation Websites with
  'Docsify.js', 'Docute', or 'Mkdocs_.
  https://altdoc.etiennebacher.com,
  https://github.com/etiennebacher/altdoc.

A BibTeX entry for LaTeX users is

  @Manual{,
    title = {altdoc: Package Documentation Websites with 'Docsify.js', 'Docute', or
'Mkdocs},
    author = {Etienne Bacher},
    year = {2023},
    note = {https://altdoc.etiennebacher.com,
https://github.com/etiennebacher/altdoc},
  }
```
